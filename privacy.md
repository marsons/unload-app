# Politique de confidentialité — Unload

Dernière mise à jour : mars 2026

Unload respecte votre vie privée.

L’application est conçue pour fonctionner principalement **en local sur votre appareil**.

## Données collectées

Unload **ne collecte aucune donnée personnelle**.

L’application :
- n’utilise pas d’analytics
- n’utilise pas de tracking
- ne revend pas de données
- ne transmet pas vos pensées à des serveurs externes

Les pensées capturées dans l’application sont stockées localement sur votre appareil via le système de stockage d’iOS.

## Notifications

Unload peut demander l’autorisation d’envoyer des **notifications locales** afin de vous rappeler de faire votre rituel lorsque des pensées sont en attente.

Ces notifications sont générées localement sur votre appareil.

## Intégrations

L’application peut exporter des tâches vers :
- **Apple Reminders**
- **Todoist**

Lorsque vous utilisez ces fonctionnalités, les données nécessaires sont envoyées directement au service concerné, conformément à sa propre politique de confidentialité.

## Mode dégradé

Dans de rares cas où le stockage local ne peut pas être chargé, l’application peut fonctionner temporairement en mode limité.

Dans ce mode, les pensées capturées **ne sont pas sauvegardées** et peuvent être perdues si l’application est fermée.
