# Unload — App Support

Ce dépôt contient les pages publiques nécessaires pour la publication de l’application **Unload** sur l’App Store.

Unload est une application iOS minimaliste conçue pour capturer rapidement les pensées qui vous traversent l’esprit, puis les traiter plus tard via un rituel simple.

**Principe :**

> Capture maintenant. Décide plus tard.

## Contenu

Ce repository héberge les pages GitHub Pages utilisées par l’application :

- Support utilisateur
- Politique de confidentialité
